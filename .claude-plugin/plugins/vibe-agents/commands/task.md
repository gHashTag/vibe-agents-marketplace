# Команда /task

## Описание

Запускает полный swarm intelligence пайплайн из 21 агента для автономного решения задачи.

## Использование

```
/task [описание задачи]
```

## Примеры

```
/task Создать систему авторизации с JWT
/task Настроить CI/CD для React приложения
/task Создать плагин для работы с MongoDB
/task Проанализировать безопасность кода
```

## Как работает

При выполнении команды `/task` автоматически запускается пайплайн:

1. **VIBE-LEAD** анализирует задачу и определяет нужных агентов
2. **VIBE-SPEC** создает детальную спецификацию (OpenAPI 3.2 + JSON Schema)
3. **VIBE-TASKER** составляет план задач с приоритетами
4. **Параллельное выполнение агентов**:
   - VIBE-TYPESCRIPT создает типы
   - VIBE-TESTER пишет TDD тесты (RED → GREEN → REFACTOR)
   - VIBE-SECURITY проводит аудит безопасности
   - VIBE-CODER реализует код
5. **VIBE-CRITIC** проводит код-ревью
6. **VIBE-SENTRY** настраивает мониторинг
7. **VIBE-ROI** анализирует эффективность
8. **VIBE-KNOWLEDGE-KEEPER** обновляет документацию

**Время выполнения:** 5-15 минут (вместо дней/недель!)

## Особенности

- ✅ **Полная автономность** - агенты работают самостоятельно
- ✅ **TDD подход** - тесты пишутся перед кодом
- ✅ **Параллельное выполнение** - максимальная скорость
- ✅ **Автоматическое исправление ошибок** - retry до успеха
- ✅ **Multi-level контроль качества** - security, types, review
- ✅ **Русская локализация** - удобно для СНГ
